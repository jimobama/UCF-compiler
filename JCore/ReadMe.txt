========================================================================
    DYNAMIC LINK LIBRARY : JCore Project Overview
========================================================================

This is the set of the libraries that carryout the underline character parsing to make the library flexible.
It contains 

IComparable interface that should implemented by any container 
Core.h That have all the libraries included into its, users should include the header file only if they want to used all the library headers
InfixExpression: That convert infix to postfix expression
SStream.h that is a container that contains symbol chracters
TokenClass: That contains token ID

This library is the building blocks for the world API library for the front-end compiler.
